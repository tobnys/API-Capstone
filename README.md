# **API Capstone project**

## Summary
This app uses the openweatherapi to display data on whatever query is sent to it. Users can easily search on the app and also check current weather for some already defined sites.

# Technology used
* **Front-end**
 * HTML5
 * CSS3
 * JavaScript
 * jQuery
